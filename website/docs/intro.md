# Welcome to AI Agent Management Platform

The AI Agent Management Platform (AMP) is a comprehensive solution for building, deploying, and managing AI agents at scale.

## What is AMP?

AMP provides enterprises with a complete control plane for AI agent lifecycle management, from development to production. Built on OpenChoreo and leveraging OpenTelemetry, it offers deep observability and governance capabilities for both internally hosted and externally deployed agents.

## Architecture

The platform consists of several integrated components:

- **Console UI** - Web-based management interface
- **Agent Manager API** - Core control plane API
- **Traces Observer** - Trace query and analysis service
- **Python Instrumentation** - Auto-instrumentation for Python agents
- **OpenChoreo Integration** - Kubernetes-native deployment platform
